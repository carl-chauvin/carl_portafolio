PortafolioProject

Tecnologias Usadas: HTML, CSS & JavaScript

El proyecto Portafolio Web se enfoca en el diseño y la estructura visual de una pagina web personal utilizando HTML, CSS y JavaScript.

Este proyecto implica la creación de una pagina web personal con una interfaz atractiva y funcional desmostrando los diferentes proyectos realizados. Se trata de desarrollar una presentación visual agradable y accesible para los usuarios, utilizando HTML para la estructura del contenido y CSS para el diseño, y JavaScript para la parte interactiva de la pagina.

Incluye la disposición de elementos como encabezados, información personal del Desarrollador y proyectos realizados, con el fin de ofrecer una experiencia de usuario intuitiva y atractiva para los visitantes del sitio.
